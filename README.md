## 📅 Study Plan & Checklist

### Phase 1: Core Material (Microsoft Learn)
- [ ] **Domain 1: SCI Concepts**
  - [ ] Describe the Shared Responsibility Model and Defense-in-Depth
  - [ ] Understand the 3 pillars of Zero Trust (*Verify explicitly, Use least privileged access, Assume breach*)
  - [ ] Master Identity concepts (Authentication vs. Authorization, Federation)
- [ ] **Domain 2: Microsoft Entra**
  - [ ] Understand Entra ID core concepts and identity types
  - [ ] Master Auth capabilities (MFA, Conditional Access, Self-Service Password Reset)
  - [ ] Review Governance features (Privileged Identity Management - PIM, Access Reviews)
- [ ] **Domain 3: Security Solutions**
  - [ ] Review Microsoft Defender for Cloud and Cloud Security Posture Management (CSPM)
  - [ ] Map out Microsoft Defender XDR (Endpoint, Office 365, Identity, Cloud Apps)
  - [ ] Define Microsoft Sentinel (Understand SIEM vs. SOAR)
- [ ] **Domain 4: Compliance Solutions**
  - [ ] Explore the Service Trust Portal & Microsoft Priva
  - [ ] Understand Microsoft Purview (Compliance Manager, Data Classification, Insider Risk, eDiscovery)

### Phase 2: Practice & Review
- [ ] Take the first official **Microsoft Practice Assessment** to establish a baseline score.
- [ ] Document all incorrect answers.
- [ ] Review flashcards/notes on renamed services (e.g., *Azure AD -> Microsoft Entra ID*).
- [ ] Retake the practice exam until consistently scoring **>85%**.
